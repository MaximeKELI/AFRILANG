Tout ce qu'AFRILANG sait faire

Vue d'ensemble complète pour utilisateurs, contributeurs et curieux.

Langage

  • Syntaxe naturelle FR / EN (dire, si, fin…)
  • Types : number, text, bool, list, map, option, result
  • POO : classes, héritage, interfaces, records, enums, unions
  • Génériques, lambdas, opérateurs surchargés
  • Match avec guards, générateurs lazy, décorateurs

Compilateur

  • Transpilation C++17 + g++ / clang++
  • Cross-compile : native, linux-x64, linux-arm64, wasm32
  • Cache de compilation, symboles debug
  • Mode sécurisé sandbox, limites réseau
  • 60+ exemples dans la suite de tests

CLI

  • run, build, check, test
  • fmt, lint, repl, doc
  • pkg add|install|sync|reindex|list --blessed
  • debug (GDB + .afr.debug.json)
  • benchmark, serve (playground)

Stdlib (14+ modules)

  • io, json, fs, http, crypto, yaml, csv, html
  • math, datetime, collections, args, path, thread
  • sql, orm, web, bigint, random, hex, url…
  • Import : import "std/json" + use json

Paquets blessed ★
