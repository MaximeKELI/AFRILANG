Philosophie

Lisibilité d'abord

Les conditions et boucles utilisent des mots : is greater than, repeat N times, for each item in list do. Le code se lit à voix haute.

Bilingue FR / EN

Même programme en dire ou say, si ou if, nombre ou number. Messages d'erreur localisables via AFRILANG_LOCALE=fr|en.

Performance réelle

Pas de VM ni d'interpréteur : transpilation + g++. Coroutines C++20 pour async/await. Debug avec mapping #line vers les sources .afr.

Comparaison syntaxique

Même logique — syntaxe AFRILANG vs style C traditionnel.

Style traditionnel
if (score >= 90) {
    printf("Excellent\n");
} else if (score >= 50) {
    printf("Pass\n");
} else {
    printf("Retry\n");
}
AFRILANG
if score is greater than or equal to 90 then
    say "Excellent"
else if score is greater than or equal to 50 then
    say "Pass"
else
    say "Retry"
end

Pipeline de compilation

fichier.afr  →  Lexer  →  Parser  →  AST  →  Analyse sémantique  →  CodeGen C++  →  g++  →  exécutable
  • Imports multi-fichiers : import "helpers.afr", import "std/io"
  • Projets avec manifeste afrilang.toml et résolution semver des dépendances
  • Cibles : native, linux-x64, linux-arm64, wasm32

Fonctionnalités du langage

  • Variables, constantes, types annotés ou inférés
  • Listes naturelles : list of 1, 2, 3, map each, filter each
  • Fonctions, lambdas, modules avec export / private
  • Classes, héritage, abstract, final, static, properties
  • Interfaces (traits) et polymorphisme
  • Génériques sur fonctions et classes
  • Enums, unions taguées, match exhaustif (statement et expression)
  • Optionnels T?, type Result, exceptions try/catch/raise
  • async function / await (C++20 coroutines)
  • FFI : extern from "lib" function …
  • Tests intégrés : blocs test "name" avec assert

Exemple complet : match + Result

status.afr
enum Status
    case Ok
    case Error with message text
end

function divide(a number, b number) returns number or error
    if b is equal to 0 then
        return error "division by zero"
    end
    return a / b
end

create r = divide(10, 2)
match r
    case error with msg then
        say "Failed: " + msg
    end
    default
        say "Result: " + r.value
    end
end
