Pourquoi AFRILANG

Lisible comme une phrase, performant comme du C++

Conçu pour l'apprentissage et la production : syntaxe naturelle, erreurs explicites, outils modernes et écosystème open source africain.

💬

Syntaxe naturelle

if age is greater than 18 then au lieu de symboles cryptiques. Mots-clés en français (dire, si, fin) ou en anglais.

⚡

Performance native

Transpilation vers C++17 puis compilation g++. Pas d'interpréteur : binaires natifs, cross-compilation linux-x64, linux-arm64 et wasm32.

🧩

Moderne et complet

POO, génériques, enums, match, traits, async/await, Result, null-safety, surcharge d'opérateurs et FFI — le tout en syntaxe lisible.

📚

Stdlib massive

I/O, JSON, HTTP, collections, graphes, stats, crypto, finance, NLP… Plus de 460 modules prêts à l'emploi via import "std/…".

🛠

Toolchain intégrée

CLI (build, test, fmt, lint), LSP 1.1, REPL, debug adapter, extension VS Code/Cursor, playground web.

🌍

Open source africain

Projet communautaire sous licence MIT. Gestionnaire de paquets, registre officiel et documentation complète pour contribuer et publier.

Exemple rapide

De la phrase au binaire en une commande

Installez le compilateur, écrivez un fichier .afr, exécutez-le. Le pipeline transpile vers C++ puis invoque g++ automatiquement.

Terminal
git clone https://github.com/MaximeKELI/AFRILANG
cd AFRILANG/build && cmake .. && cmake --build .
./afrilang run examples/hello.afr
./afrilang init mon_app && cd mon_app
afrilang build && afrilang run src/main.afr
examples/oop.afr
class Animal
    function speak()
        say "..."
    end
end

class Dog extends Animal
    public field name text

    function init(aName text)
        set this.name = aName
    end

    function speak()
        say name + " says Woof!"
    end
end

create rex = new Dog("Rex")
rex.speak()
