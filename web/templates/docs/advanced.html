Fonctionnalités avancées

Génériques, pattern matching, async/await, exceptions, FFI, tests intégrés et interface graphique SDL2.

Génériques

function swap(a T, b T) returns list of T
    return list of b, a
end

function max(a T, b T) returns T
    if a is greater than b then
        return a
    end
    return b
end

say max(3, 7)
say max("a", "z")

Pattern matching

enum Status
    case Ok
    case Error with message text
end

create e = Status.Error with "timeout"

match e
    case Ok then
        say "success"
    end
    case Error with msg then
        say "Error: " + msg
    end
end

Tous les cas doivent être couverts ou un default doit être présent. Liaison de payload : case Error with msg then.

Match en expression

create label = match e
    case Ok then "success"
    end
    case Error with msg then msg
    end
    default "unknown"
end

Async / await

Nécessite C++20 et les coroutines (-std=c++20 -fcoroutines).

import "std/async"

async function fetchData() returns task text
    await sleep(100)
    return "data loaded"
end

async function main() returns task
    create result = await fetchData()
    say result
end

Modules associés : std/async, std/http pour httpGetAsync.

Exceptions

try
    if balance is less than amount then
        raise "insufficient funds"
    end
    set balance = balance - amount
catch error e
    say "Caught: " + e
end

FFI (Foreign Function Interface)

extern from "m" function sin(x number) returns number
extern from "m" function cos(x number) returns number

say sin(3.14159 / 2)

Le compilateur lie les bibliothèques système (libm, pthread, curl, etc.).

Tests intégrés

function add(a number, b number) returns number
    return a + b
end

test "addition works"
    assert add(2, 3) is equal to 5
    assert add(-1, 1) is equal to 0
end

Les blocs test s'exécutent au lancement du programme ou via afrilang test.

Mode explain (pédagogique)

explain say "Hello"
explain create x = 42

Ou en CLI : afrilang explain fichier.afr — décrit chaque ligne en langage naturel.
