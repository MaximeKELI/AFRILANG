Exemple : lecture de fichier JSON

import "std/io"
import "std/json"

create raw = readFile("config.json")
create doc = parse(raw)
say getString(doc, "name")
say getNumber(doc, "version")

Catalogue tiered (463 modules)

La stdlib est organisée en trois niveaux de complexité :

Niveau simple — import "std/nom"

151 bibliothèques légères : math (sqrtlib, prime, fib), texte (upper, slug, email), conversions (celsius, hex, currency), géométrie (circle, triangle), finance basique (loan, tax), utilitaires (semver, uuid, calendar).

Niveau medium — import "std/m/nom"

102 bibliothèques : analyse texte (levenshtein, ngrams), algos listes (listsort, listquantile), dates (datecalc, timezone2), parsing (jsonparse2, csvparse2), algèbre linéaire (matrix2, matmul2), finance (npv2, irr2).

Niveau complex — import "std/c/nom"

210 bibliothèques avancées :

  • Graphes — BFS, DFS, Dijkstra, Floyd-Warshall, MST, max-flow
  • Stats & ML — régression, ANOVA, k-means, SVM, PCA, TF-IDF, naive Bayes
  • Crypto — SHA-256, HMAC, AES-lite, Base64
  • Compression — RLE, Huffman, LZW
  • Parsers — JSON, XML, YAML, TOML, INI, CSV, HTML
  • Finance — Black-Scholes, VaR, Sharpe, Greeks
  • Signal — FFT, convolution, filtres
  • NLP — stemming, sentiment, simhash
  • Game AI — minimax, alpha-beta, A*, MCTS
Documentation détaillée — voir dans le dépôt : docs/STDLIB_SIMPLE.md, STDLIB_MEDIUM.md, STDLIB_COMPLEX.md.

Opérations listes intégrées

Sans import, le langage fournit :

create nums = list of 1, 2, 3, 4, 5
create doubled = map each n in nums with n * 2
create evens = filter each n in nums where n is greater than 2
create sum = reduce nums from 0 with a, b => a + b
