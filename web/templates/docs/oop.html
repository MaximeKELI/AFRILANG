Programmation orientée objet

Classes, héritage, encapsulation, interfaces, records et classes génériques — la POO complète d'AFRILANG avec une syntaxe lisible.

Classes et instances

class Animal
    function speak()
        say "..."
    end
end

class Dog extends Animal
    public field name text

    function init(aName text)
        set this.name = aName
    end

    function speak()
        say name + " says Woof!"
    end
end

create rex = new Dog("Rex")
rex.speak()

Encapsulation

class BankAccount
    private field balance number

    function init(initial number)
        set this.balance = initial
    end

    public function deposit(amount number)
        set this.balance = this.balance + amount
    end

    public function getBalance() returns number
        return this.balance
    end
end

Modificateurs : public, private, protected.

Héritage et super

class Vehicle
    public field brand text

    function init(brand text)
        set this.brand = brand
    end

    function describe() returns text
        return brand
    end
end

class Car extends Vehicle
    public field model text

    function init(brand text, model text)
        super(brand)
        set this.model = model
    end

    function describe() returns text
        return super.describe() + " " + model
    end
end

Membres static, abstract et final

abstract class Shape
    abstract function area() returns number
end

final class Circle extends Shape
    public field radius number

    function init(r number)
        set this.radius = r
    end

    function area() returns number
        return 3.14159 * radius * radius
    end
end

class Counter
    static field count number = 0

    static function increment()
        set Counter.count = Counter.count + 1
    end
end

Properties et destructeur

class Person
    property text name end
    property number age end

    destroy
        say "Goodbye, " + name
    end
end

Interfaces (traits)

interface Speakable
    function speak()
end

interface Drawable
    function draw()
end

class Robot implements Speakable
    function speak()
        say "Beep"
    end
end

create speakers list of Speakable = list of new Robot()
for each s in speakers do
    s.speak()
end

Polymorphisme

create pet Animal = new Dog("Rex")
pet.speak()   # dispatch dynamique

Records

Structs légers pour les données immuables ou semi-structurées :

record Point
    field x number
    field y number
end

create p = Point with 10, 20
say p.x

Classes génériques

class Box
    public field value T

    function init(v T)
        set this.value = v
    end

    function get() returns T
        return this.value
    end
end

create box = new Box(42)
say box.get()

Surcharge d'opérateurs

class Vector
    public field x number
    public field y number

    operator + (other Vector) returns Vector
        return new Vector(this.x + other.x, this.y + other.y)
    end
end

Opérateurs supportés : +, -, *, /, ==, !=, <, >.
