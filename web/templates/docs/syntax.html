Syntaxe de base

Variables, entrées/sorties, conditions, boucles, listes, maps et fonctions — les fondations du langage AFRILANG en syntaxe naturelle.

Affichage et entrée

say "Hello, world!"          # alias FR : dire
ask "Votre nom ?" into name
say "Bonjour, {name}!"       # interpolation de chaîne

Variables et constantes

create x = 42                    # inférence
create name text = "AFRILANG"    # type explicite
create const MAX = 100           # constante
set x = x + 1                    # réassignation

Conditions

Les comparaisons utilisent des mots-clés au lieu de symboles :

create age number = 21

if age is greater than 18 then
    say "Majeur"
else if age is equal to 18 then
    say "Juste majeur"
else
    say "Mineur"
end

Opérateurs disponibles : is equal to, is not equal to, is greater than, is less than, is greater than or equal to, is less than or equal to.

Boucles

# Répéter N fois
repeat 5 times
    say "tick"
end

# Tant que
create count number = 0
while count is less than 10 do
    say count
    set count = count + 1
end

# For each (listes et maps)
create nums = list of 10, 20, 30
for each n in nums do
    say n
end

# For numérique
for i from 1 to 10 step 2 do
    say i
end

# Contrôle de boucle
stop    # break
skip    # continue

Listes

create nums = list of 1, 2, 3
create empty = empty list number
create bracket = [10, 20, 30]

say nums at 0              # indexation
set nums at 1 = 99
say length of nums
add 4 to nums

# Opérations fonctionnelles (sans import)
create doubled = map each n in nums with n * 2
create evens = filter each n in nums where n is greater than 2
create total = reduce nums from 0 with a, b => a + b

Maps

create m = map of "a" to 1, "b" to 2
create scores map text to number = empty map text to number

set scores at "alice" = 95
say scores at "alice"

for each key, value in scores do
    say key + ": " + value
end

Fonctions

function add(a number, b number) returns number
    return a + b
end

function greet(name text, prefix text = "Hello") returns text
    return prefix + ", " + name
end

say add(3, 4)
say greet("World")

# Lambda
create double = function(x number) returns number
    return x * 2
end

Modules

module Math
    export function square(x number) returns number
        return x * x
    end

    private function helper() returns number
        return 1
    end
end

use Math
say Math.square(5)

Imports

import "helpers.afr"       # fichier local
import "std/io"              # module standard
import "pkg/math/math.afr"   # paquet externe
